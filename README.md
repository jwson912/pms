# Vue/React 연구회 활동

Vue 3, Vite, Pinia, Vue Router로 만든 학습용 주차관리 시스템으로 별도 DB서버 없이 브라우저 `localStorage`만 사용

## 배포
- Vercel : 프론트엔드 중심 웹 서비스를 쉽게 배포·운영할 수 있게 해주는 클라우드 플랫폼
- URL : https://jw-pms.vercel.app

## 실행

```bash
npm install
npm run dev
```

## 주요 기능

- 회원가입/로그인: 아이디 기반 계정 정보를 브라우저 `localStorage`의 `parking.user.profiles` 키에 저장
- 대시보드: 현재 주차, 잔여 공간, 오늘 입·출차 및 매출
- 입차등록: 차량 번호 정규화, 중복·최대 100대 검사
- 출차처리: 주차 시간과 요금 자동 계산, 사용자/관리자 공통 접근, 할인쿠폰 적용, 출차 완료 후 10분 이내 출차 안내
- 주차현황: 차량 번호·상태 검색, 실시간 주차 시간 표시, 기록 삭제
- 공지사항: 사용자/관리자 공통 접근 게시판
- 사용자 관리: 관리자 전용 사용자 목록/권한/삭제 관리
- 내정보: 내 계정으로 등록된 입차 차량과 차량번호판 이미지 표시
- 설정: 수용량 및 요금 정책 변경

요금 기본값은 30분 1,000원, 이후 10분당 300원, 1일 최대 15,000원입니다.

## 데이터 저장 위치

모든 데이터는 현재 브라우저의 `localStorage`에 저장

- 주차 기록: `parking.records`
- 요금/수용량 설정: `parking.settings`
- 현재 로그인 세션: `parking.admin.session`
- 사용자 목록/권한/회원가입 계정: `parking.user.profiles`
