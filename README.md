# Vue.js 소규모 주차관리 시스템 MVP

Vue 3, Vite, Pinia, Vue Router로 만든 학습용 주차관리 시스템입니다. 별도 서버 없이 브라우저 `localStorage`만 사용합니다.

## 실행

```bash
npm install
copy .env.example .env
npm run dev
```

기본 관리자 계정은 아래 1개입니다.

- `admin@miracom-inc.com` / `admin123`

## 주요 기능

- 대시보드: 현재 주차, 잔여 공간, 오늘 입·출차 및 매출
- 입차: 차량 번호 정규화, 중복·최대 100대 검사
- 출차: 주차 시간과 요금 자동 계산
- 현황: 차량 번호·상태 검색, 실시간 주차 시간 표시, 기록 삭제
- 설정: 수용량 및 요금 정책 변경
- 회원가입/로그인: 아이디 기반 계정 정보를 브라우저 `localStorage`에 저장
- 공지사항: 사용자/관리자 공통 접근 게시판
- 사용자 관리: 관리자 전용 사용자 목록/권한/삭제 관리
- 내정보: 내 계정으로 등록된 입차 차량과 차량번호판 이미지 표시
- 출차: 사용자/관리자 공통 접근, 할인쿠폰 적용, 출차 완료 후 10분 이내 출차 안내

요금 기본값은 30분 1,000원, 이후 10분당 300원, 1일 최대 15,000원입니다.

## 데이터 저장 위치

모든 데이터는 현재 브라우저의 `localStorage`에 저장됩니다.

- 주차 기록: `parking.records`
- 요금/수용량 설정: `parking.settings`
- 현재 로그인 세션: `parking.admin.session`
- 사용자 목록/권한: `parking.user.profiles`
